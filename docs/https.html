Secure application with HTTPS

How to add HTTPS

  1. Add this line to your <main> tag within your project's .gxml file. If there is no <main> tag, add one to the <gos> tag of your file.
  2. Line to add : go http.ListenAndServeTLS(":443", "/path-to-crt-file", "/path-to-key-file", nil)
  3. If you have multiple certificate files, create a new file. Append the certificates you need to this new file.
