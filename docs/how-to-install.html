How to install

Run the following commands to install GopherSauce :

go get github.com/cheikhshift/gos
gos deps

Generate project

Use the --make command to create a new project in your current directory. For Example :

gos --make

Handle GopherSauce Dependencies

The application generated by GopherSauce has certain dependencies. Please run the command below to download them before attempting to run or export your application. (Assuming $GOPATH is set)

gos deps
