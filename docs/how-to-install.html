How to install

go get github.com/cheikhshift/gos
go install github.com/cheikhshift/gos
gos deps

Generate project

Use the --make command to create a new project in your current directory. For Example :

$GOPATH/bin/gos --make

OR -

Handle Go Server Dependencies

The application generated by Go Server has certain dependencies. Please run the command below to download them before attempting to run or export your application. (Assuming $GOPATH is set)

$GOPATH/bin/gos deps
