Middleware

Execute blocks of go code when a request path contains your supplied string. If you supplied a path of sample/, your block would run on request with sample paths : /any/string/sample/code/goes or /another/path/code/sample/.

How-to

Within your .gxml file add an <end> tag with type f. Your file should look like this : 

   <gos>
     <endpoints>
       <end type="f" path="/path/to/match">
       /* Go code */
     </end>
    </endpoints>
  </gos>

Access control.

Within your <end> tag with type f return true to block a request. Do not return anything to keep the request going.

Redirection

Prior to returning true, you may redirect your requests with the following Go lang statement : http.Redirect(w,r,"/path-to-redirect",302). Update "/path-to-redirect" to your new page.

Execution Order.

Your Middleware paths will be executed from top to bottom as listed in your .gxml file.
