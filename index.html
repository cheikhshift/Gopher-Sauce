Golang Server

Golang Server is a web application development tool.
It aims to give you the best experience during development. Golang Server features:

Agile — During your development and to your clients' requests during production.

Tools — Build faster with ready-to-use extensions.

Ready for scale/Docker — Each project is generated with a Dockerfile.

Simple tracing — Golang server supports Opentracing.

How does it work?

Template pipeline creation.

Go Server simplifies the task of extending your Go lang {{ templates' }} pipelines.
The following snippet will create a pipeline {{Pipeline}} available to all your Go server templates. 

  
          <gos>
            ...
              <methods>
                <method name="Pipeline" return="string">
                    return "string"
                </method>
              </methods>
            ...
          </gos>
       
  
          <gos>
            ...
                <endpoints>      
                  <end path="/login" type="GET" >
                      fmt.Println("Login attempt -->")
                  </end>
                </endpoints>
            ...
          </gos>

Building web services

Go Server simplifies the task of assigning a function to a server path. The prior snippet will create a web service at path /login. The endpoint will log to your console via Go lang package fmt

DevOps

Request <method> and <end> tags from your favorite developers. The following snippet demonstrates importing a Go-Server package. 

  
          <gos>
            ...
                <import src="outsourced/pkg/gos.gxml" />
            ...
          </gos>

Install Go Server

Run the following commands with your terminal :


    go get github.com/cheikhshift/gos
    gos deps

Quick Start your project


    mkdir new-project
    cd new-project
    gos --make
