Build a website with templates.

Use your Golang libraries with your Web app.

Build MVCs with Go

Create a new file in your template directory named test

$ touch tmpl/test.tmpl

Contents of file tmpl/test.tmpl

<div class="header">
 <img src="logo.png" style="float:left" width="60" />
<a href="/Link to home" >Home</a> 
<a href="/Link to Page" >Link</a>
</div>

Link template in gos.gxml

<gos>  
 ...
    <templates>
        ...
         <template name="Header" tmpl="test" struct="" /> 
    </templates>
</gos>

Using template in Gos project

Create a new file within the web folder.

$ touch web/page.tmpl

Contents of page.tmpl : 

<html>
{{ Header }}
</html>

Test page at url localhost:8080/page

Add models to your templates to create MVC's with Go.

Create new method in gos.gxml.

 <gos>
        <methods>
       ...
        </methods>
        ...
    </gos>

Link method to endpoint

 <gos>
    ...
    <endpoints>
           <end path="/index/api" type="POST" >
                  /*response - API response var */
                  response = mResponse(Model{key:data,field:0})
           </end>
    </endpoints>
    ...
</gos>

Test endpoint at url localhost:8080/index/api
